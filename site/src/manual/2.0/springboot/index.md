# buession-canal-springboot 参考手册


buession canal springboot 支持。

### 安装

```xml
<dependency>
    <groupId>com.buession.canal</groupId>
    <artifactId>buession-canal-springboot</artifactId>
    <version>x.x.x</version>
</dependency>
```

该模块主要为对 springboot 支持。
